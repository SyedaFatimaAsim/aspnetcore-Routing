# ASP.NET Core Routing Demo

## 📌 Description
This project demonstrates routing in ASP.NET Core using UseEndpoints.  
It includes examples of route parameters, constraints, and custom responses.

---

## 🚀 Features
- Get all countries
- Get country by ID
- Return 404 for invalid country
- Route constraints (int, min, max, range)

---

## 🛠 Technologies Used
- ASP.NET Core
- C#

---

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/SyedaFatimaAsim/aspnetcore-Routing.git

2. Open in Visual Studio

3. Run the project

---

## 📌 API Endpoints

### Get all countries
GET /countries

### Get country by ID
GET /countries/{id}

Example:
GET /countries/3 → United Kingdom  
GET /countries/6 → 404 [No Country]
